# Facebook-Marketplace-Redesign
UX/UI redesign project to improve Facebook Marketplace

## Project Overview
This project is a UX/UI redesign of Facebook Marketplace, aiming to create a safer, more organized, and user-friendly platform.
Facebook Marketplace is widely used, but users often face problems such as fraud, lack of seller information, and difficulty in finding products.
The redesign focuses on solving these issues by improving product presentation, seller transparency, search functionality, and overall user experience.

## Project Idea

The main idea behind this project is to redesign Facebook Marketplace to increase user trust and simplify the buying/selling process.
Key goals include:
 • Clear display of product and seller information
 , Inclusion of reviews and ratings
 , Organized and easy to navigate layout
 , Enhanced search and filtering tools

 ## This redesign improves:
 • User trust
 , Ease of navigation
 ,Product discovery
 , Overall buying and selling experience
